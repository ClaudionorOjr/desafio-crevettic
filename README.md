# Crevettic

> Desafio de código proposto para vaga no projeto Crevettic

## Tecnologias

- Axios
- Formik
- Material UI
- React
- React-number-format
- Redux
- UUIDV4
- TypeScript
- Yup

## Getting Started

```bash
# Instalação das dependências
npm install

# Executar projeto
npm run dev
```
